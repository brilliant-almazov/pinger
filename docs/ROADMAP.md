# Pinger Roadmap

## Текущая версия (v1.0)

- [x] Menu bar с live ping
- [x] Несколько таргетов (Google DNS, Cloudflare, custom)
- [x] Статус по каждому таргету
- [x] Настройки (интервал, порог, таргеты)
- [x] Pause/Resume
- [x] Tooltip с детализацией

## Ближайшие улучшения (v1.1)

- [ ] Launch at Login
- [ ] Уведомления при потере/восстановлении соединения
- [ ] Выбор иконки приложения в настройках
- [ ] Звуковой сигнал при проблемах
- [ ] Локализация (EN/RU)

## Расширенные фичи (v2.0)

- [ ] График латенси за последний час/день
- [ ] Статистика: uptime %, средний ping, потери пакетов
- [ ] Экспорт логов (CSV/JSON)
- [ ] Несколько профилей настроек
- [ ] Виджет для Notification Center

## Интеграции (v3.0)

- [ ] Уведомления в Telegram
- [ ] Webhook при событиях
- [ ] API для интеграции с другими приложениями
- [ ] Shortcuts/Automator поддержка

## Монетизация (если App Store)

### Бесплатная версия
- 2 таргета максимум
- Базовые настройки
- Без графиков

### Pro версия ($2.99)
- Неограниченные таргеты
- Графики и статистика
- Экспорт логов
- Уведомления в Telegram
- Приоритетная поддержка
